# Snake Game

Ein einfaches Snake-Spiel, das in JavaScript geschrieben ist und im Terminal ausgeführt wird. Das Spiel ermöglicht es dem Spieler, die Schlange zu steuern, Futter zu sammeln und die Punkte zu zählen. Das Ziel ist es, einen neuen Highscore zu erreichen.
